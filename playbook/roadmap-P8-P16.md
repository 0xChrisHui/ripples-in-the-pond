## Phase 8：水塘视觉重设计

**目标：把首页从"星空视觉"改成"水塘视觉"。**

主要做：

- 首页视觉语言从星空 / 宇宙感，转向水塘 / 涟漪 / 水面感
- 小球仍先保持现有数量，不在本 Phase 改成 35
- 优先解决产品气质问题：
  - 更像 Ripples in the Pond
  - 更有水面、涟漪、池塘、漂浮感
  - 弱化星球 / 宇宙 / 星系感
- 先把首页主视觉调对

## Phase 13：混音系统 + RemixNFT

**目标：上线新的混音玩法。**

主要做：

- 基于用户钱包地址 + 唱片 hash 生成专属节拍
- 英文字母 / 数字转节拍参数
- 新增混音界面
- 接入音乐家 mp3 音轨
- 用户选择哪些音轨播放
- 一键合成 mp3
- 上传 Arweave
- 用新合约铸造新的 **RemixNFT**
- 用户可见名暂定：**混音唱片**

**算法决策（2026-06-04）**：专属节拍 = 钱包地址 + 唱片哈希 → 字母/数字转参数，默认不上链；仅在用户一键铸造时才上链（合成 mp3 → Arweave → 新合约）；不做草稿；音乐家音轨是独立 mp3，等混音界面上线后艺术家提供。
