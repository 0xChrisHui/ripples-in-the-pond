## 总则

**目标**:把定稿的 R1–R9 落地——**深化 GL 水塘的视觉与手感**(深度感 / 光感 / 交互顺滑 / 立体)。

**范围**:只在 **/test1**(+ 实验页 /test2)动;**不碰共享 SVG / 首页**(`/`、`/test` 零变化);共享 `sphere-config` 不删。

**红线(全线适用)**:球**不染色**(始终原色)/ 水下**不压黑**(亮度地板)/ 球**不缩放**(距离恒定)/ 浅塘**不虚化**。深度差异只走 折射 + 月光 + 涟漪 + 投影。

**与其他线**:K = 视觉/手感深化(沙盒内);J = 生产迁移上首页(并行/后续);R9 漂浮元素本质是 I 线 **I3 新组件**,在此一并规划。

**执行模式**:每步一闭环,**⏸ 末尾停等浏览器验收**(桌面可验为主)。

## 依赖与建议顺序

- **K3(深度三层模型)是核心**,K4(投影)依赖它;其余(K1/K2/K5/K6/K7/K8)相对独立、可调序。
- 建议序:小而基础(**K1 正圆 → K2 丝滑**)→ 核心深度(**K3** + 顺手修 R4)→ 依赖项(**K4 投影**)→ 独立手感(**K5 光照 / K6 缩放**)→ 实验/开放(**K7 /test2 / K8 漂浮元素**)。
- **K7(/test2)可提前**:先做能"感受参考",反哺 K5/K6 的目标观感。

## K1 — 涟漪改正圆（R2）⏸

- 📦 范围:涟漪/滴水的距离计算(`water/spike/ripple-spike-shaders.ts` simFrag 滴水 + `water/ripple-feed.ts` drop)。
- 根因:高度场 256² 是方的、屏幕是宽屏;距离用 UV(0–1 归一)算 → UV 圆被横向拉成椭圆。
- 做什么:滴水/涟漪距离按屏幕**宽高比(aspect)校正**(UV 距离乘 aspect),UV 圆在宽屏上仍是正圆。只改"距离度量",不动 sim 数学。
- 验收(⏸):点/划水,涟漪是正圆(各方向半径一致)。

## K2 — 鼠标划水更丝滑（R3）⏸

- 📦 范围:指针喂涟漪(`water/WaterDistort.tsx` pointermove → pending drops)。
- 做什么:**路径插值**——上帧↔当前指针位置之间按距离补几个小滴(点连成线);时间节流改**按位移触发** + 强度可随速度;配合调小 dropMove。快速划也连续、不"咚咚"。
- 验收(⏸):快速划水是连续水痕,无离散颗粒/高频点击感。

## K3 — 深度三层模型 + 调制效果（R5,含修 R4）⏸ 【核心】

- 📦 范围:深度因子 + 三带 + 折射/月光/涟漪按深度渐变;统一三个消费方(`spheres/SphereInstances.tsx` / `overlay/SphereOverlay.tsx` / `water/WaterDistort.tsx`),沿用 H5 `sphere-motion` 的 displayZ。
- 做什么:
  - **带符号深度** `s = displayZ − 水位`,分三带:**空中(s>0)** 清晰悬空(投影见 K4)/ **贴面(s≈0)** 局部接触漩涡(参考 per-flower swirl)/ **水下(s<0)** 按 `d = clamp(−s/塘深, 0, 1)` 渐变。
  - 三曲线:折射 `R·d^a`(近轻深重 + clamp 防糊)、月光 `M·(1−d)^b`(近强深弱)、涟漪响应 `(1−d)^c`(近灵敏深钝);指数/上限进参数板。
  - **统一可见性 = 修 R4**:三个消费方读同一 d(标题淡出 / 球 dim·扭曲 / 命中)一致;浮沉时连续变,消除"标签闪而球不动";标题不透明度对 d 加阻尼/平滑去闪。
- 依赖:无(K4 前置)。
- 验收(⏸):近球轻折射、深球重折射(有上限不糊);浅球月光强、深球弱;浮沉时球与标签一致连续、无闪;贴面球有接触漩涡。

## K4 — 浮出水面的球投影（R8）⏸

- 📦 范围:空中带(s>0)球在水面投柔影(DOM/GL overlay 或合成层)。
- 做什么:浮出球下方水面投**柔影**(可选淡反射),大小/柔度随空中高度 s(越高越大越柔)。配合 H5 播放浮出焦点球的"悬空"立体感。
- 依赖:K3(空中带 s)。
- 验收(⏸):播放浮出的球有"悬空 + 水面投影"3D 感;影随浮高变化。

## K5 — 水面月光焦散光照（R7）⏸

- 📦 范围:水面合成 shader(`water/water-distort-shaders.ts` compositeMaskFrag)加光照层。
- 做什么:从高度场梯度加廉价光照(借参考 #1,翻**夜塘月光冷白版**):① 漫反射(坡面微提亮)② 波峰高光(增强现有月光 spec)③ **缓慢游走的月光池 + 飘移光带**(焦散感,静止也活)。冷白、不压亮、不彩色;强度进参数板。
- 红线:不调亮整体、不上色。
- 验收(⏸):水面有月光流动/焦散感,从"哑玻璃"变"活水";静止时也有缓慢光动。

## K6 — 水面深度缩放（水位联动）（R1）⏸

- 📦 范围:合成 pass 按水位缩放高度场采样(绕中心);落地占位的「缩放」滚轮模式但**只缩水层**。
- 做什么:水位 → 水面贴图缩放(升=放大溢出 / 降=缩小露更多);连续 + 缓动;**正在进行的涟漪一起缩放**;**球不缩放**。缩小露边 → 高度场 repeat 平铺。幅度小(浅塘 ~0.9×–1.2×)进参数板。合并现有「水位/缩放」互斥模式为"水位即缩放"。
- 红线:球不缩放(距离恒定)。
- 验收(⏸):升水位水面放大、降缩小露更多;球大小不变;进行中涟漪也缩放;无露边。

- 做什么:夜塘水面漂轻盈元素,随涟漪轻漂、被球/水波推开。首批做哪种(花瓣?落叶?光点?)、数量/行为 **由用户定**。
