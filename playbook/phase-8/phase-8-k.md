## 总则

**目标**:把定稿的 R1–R9 落地——**深化 GL 水塘的视觉与手感**(深度感 / 光感 / 交互顺滑 / 立体)。

**范围**:只在 **/test1**(+ 实验页 /test2)动;**不碰共享 SVG / 首页**(`/`、`/test` 零变化);共享 `sphere-config` 不删。

**红线(全线适用)**:球**不染色**(始终原色)/ 水下**不压黑**(亮度地板)/ 球**不缩放**(距离恒定)/ 浅塘**不虚化**。深度差异只走 折射 + 月光 + 涟漪 + 投影。

**与其他线**:K = 视觉/手感深化(沙盒内);J = 生产迁移上首页(并行/后续);R9 漂浮元素本质是 I 线 **I3 新组件**,在此一并规划。

**执行模式**:每步一闭环,**⏸ 末尾停等浏览器验收**(桌面可验为主)。

## 依赖与建议顺序

- **K3(深度三层模型)是核心**,K4(投影)依赖它;K1/K2/K5/K6/K7 相对独立、可调序。
- **K8–K11 = 新组件群(本质 I3,2026-06-18 增补)**:K8 漂浮微光层 / K9 水生植物 / K10 可见塘底 / K11 月光倒影 —— 都**跟 K6 缩放联动充当"参照系"**,让缩放/纵深可感,故**排在 K6 之后**。其中 **K8 合并了原 R9 漂浮元素 + 抽象微光点 + K6 缩放参照**(同一层挂"缩放联动 + 漂浮/推"两种行为)。
- 建议序:**K1 正圆 → K2 丝滑 → K3(+修 R4) → K4 投影 → K5 光照 / K6 缩放 → K7 /test2 → K8 漂浮微光(K6 参照) → K9 植物 → K10 塘底 → K11 倒影**。
- **K7(/test2)可提前**:先做能"感受参考",反哺 K5/K6 的目标观感。

## K1 — 涟漪改正圆（R2）⏸

- 📦 范围:涟漪/滴水的距离计算(`water/spike/ripple-spike-shaders.ts` simFrag 滴水 + `water/ripple-feed.ts` drop)。
- 根因:高度场 256² 是方的、屏幕是宽屏;距离用 UV(0–1 归一)算 → UV 圆被横向拉成椭圆。
- 做什么:滴水/涟漪距离按屏幕**宽高比(aspect)校正**(UV 距离乘 aspect),UV 圆在宽屏上仍是正圆。只改"距离度量",不动 sim 数学。
- 验收(⏸):点/划水,涟漪是正圆(各方向半径一致)。

## K2 — 鼠标划水更丝滑（R3）⏸

- 📦 范围:指针喂涟漪(`water/WaterDistort.tsx` pointermove → pending drops)。
- 做什么:**路径插值**——上帧↔当前指针位置之间按距离补几个小滴(点连成线);时间节流改**按位移触发** + 强度可随速度;配合调小 dropMove。快速划也连续、不"咚咚"。
- 验收(⏸):快速划水是连续水痕,无离散颗粒/高频点击感。

## K3 — 深度三层模型 + 调制效果（R5,含修 R4）⏸ 【核心】

- 📦 范围:深度因子 + 三带 + 折射/月光/涟漪按深度渐变;统一三个消费方(`spheres/SphereInstances.tsx` / `overlay/SphereOverlay.tsx` / `water/WaterDistort.tsx`),沿用 H5 `sphere-motion` 的 displayZ。
- 做什么:
  - **带符号深度** `s = displayZ − 水位`,分三带:**空中(s>0)** 清晰悬空(投影见 K4)/ **贴面(s≈0)** 局部接触漩涡(参考 per-flower swirl)/ **水下(s<0)** 按 `d = clamp(−s/塘深, 0, 1)` 渐变。
  - 三曲线:折射 `R·d^a`(近轻深重 + clamp 防糊)、月光 `M·(1−d)^b`(近强深弱)、涟漪响应 `(1−d)^c`(近灵敏深钝);指数/上限进参数板。
  - **统一可见性 = 修 R4**:三个消费方读同一 d(标题淡出 / 球 dim·扭曲 / 命中)一致;浮沉时连续变,消除"标签闪而球不动";标题不透明度对 d 加阻尼/平滑去闪。
- 依赖:无(K4 前置)。
- 验收(⏸):近球轻折射、深球重折射(有上限不糊);浅球月光强、深球弱;浮沉时球与标签一致连续、无闪;贴面球有接触漩涡。

## K4 — 浮出水面的球投影（R8）⏸

- 📦 范围:空中带(s>0)球在水面投柔影(DOM/GL overlay 或合成层)。
- 做什么:浮出球下方水面投**柔影**(可选淡反射),大小/柔度随空中高度 s(越高越大越柔)。配合 H5 播放浮出焦点球的"悬空"立体感。
- 依赖:K3(空中带 s)。
- 验收(⏸):播放浮出的球有"悬空 + 水面投影"3D 感;影随浮高变化。

## K5 — 水面月光焦散光照（R7）⏸

- 📦 范围:水面合成 shader(`water/water-distort-shaders.ts` compositeMaskFrag)加光照层。
- 做什么:从高度场梯度加廉价光照(借参考 #1,翻**夜塘月光冷白版**):① 漫反射(坡面微提亮)② 波峰高光(增强现有月光 spec)③ **缓慢游走的月光池 + 飘移光带**(焦散感,静止也活)。冷白、不压亮、不彩色;强度进参数板。
- 红线:不调亮整体、不上色。
- 验收(⏸):水面有月光流动/焦散感,从"哑玻璃"变"活水";静止时也有缓慢光动。

## K6 — 水面深度缩放（水位联动）（R1）⏸

- 📦 范围:合成 pass 按水位缩放高度场采样(绕中心);落地占位的「缩放」滚轮模式但**只缩水层**。
- 做什么:水位 → 水面贴图缩放(升=放大溢出 / 降=缩小露更多);连续 + 缓动;**正在进行的涟漪一起缩放**;**球不缩放**。缩小露边 → 高度场 repeat 平铺。幅度小(浅塘 ~0.9×–1.2×)进参数板。合并现有「水位/缩放」互斥模式为"水位即缩放"。
- 红线:球不缩放(距离恒定)。
- 验收(⏸):升水位水面放大、降缩小露更多;球大小不变;进行中涟漪也缩放;无露边。

## K7 — /test2 参考水实验页（R6）⏸ 【实验,可提前】

- 📦 范围:新建 `app/test2/`(noindex,抄 /test1 layout),移植 `references/flower-water-ripples` 的水(water.jpg + 涟漪 sim + 折射合成)作背景 + 叠球(复用 `useGlSim`/`SphereOverlay`)。
- 做什么:水位卡中段(**50% 球水上 / 50% 水下**),直观感受球嵌在参考那种明亮水里的观感;为 K5/K6/K3 提供手感实据。**纯实验、不入生产**。

## K8 — 水面漂浮层（微光/花瓣：缩放参照 + 漂浮/推）（R9 + K6 参照,= I3 新组件）⏸ 【开放】

> **2026-06-18 合并定案**:把"K6 缩放参照"与原"K8 漂浮元素"合成**一个组件** —— 同一层漂浮物挂两种行为:① 跟 K6 缩放联动(当缩放参照) ② 随涟漪轻漂 + 被推(漂浮手感)。本质同一类东西(微光/花瓣都是"水面漂浮物"),不做两套。

- 📦 范围:新漂浮组件(一层点精灵/粒子,**微光为主、可切花瓣 style**),挂水面层;与 K6 缩放、H4/K2 涟漪、指针/球推力联动;控制台开关 + 密度/style/强度参数。
- 做什么:夜塘水面漂一层**抽象微光点**(冷调浮尘/浮游微光;style 可切花瓣)。挂两种行为:
  - **① 缩放参照(解决 K6"看不出缩放")**:位置/大小绕画面中心随水位放缩 —— 水位升整片收拢变大、降则散开变小;**密集小点的"光流"扩散收拢一眼可感缩放**(花瓣疏大→参照弱、偏氛围)。**缩放为主、漂移克制**,免乱漂糊掉缩放感。
  - **② 漂浮/推(原 R9 手感)**:随涟漪轻漂、被指针/球推开("推水")。机制(参考 `updatePetals`):被水面高度梯度/指针速度推 `v += grad·k` + 阻尼 + 自转(参考水波本身对称、方向感全靠被推的漂浮物)。水流来源:v1 指针速度近似(零回读)、v2 真高度梯度(降采样回读)。
- **路线 A(定)**:独立漂浮层,**不动球/不碰布局/不碰红线**;微光为主(契合"抽象" + 当 K6 参照最强),花瓣作 style 备选。
- 依赖:**K6**(缩放变换——做参照需联动);其余独立。
- 验收(⏸):开漂浮层 → 升降水位时整片微光随之缩放(一眼读出 K6 缩放);划水/拖球时被推轻漂;不抢球、夜塘冷调、性能正常。
- 归属:本质 I 线 **I3 新组件**。

## K9 — 水面水生植物（睡莲叶/荷叶 + 芦苇/水草）⏸ 【开放】

- 📦 范围:水面植物层(睡莲叶/荷叶平铺 + 边缘芦苇/水草丛);跟 K6 缩放联动;控制台开关 + 数量/分布参数。
- 做什么:俯视**睡莲叶圆片**平铺水面(墨绿冷调、半透),**边缘芦苇/水草丛**暗示池塘边界 = 框构图 + 纵深;跟缩放 UV 一起放缩(比微光更"有形"的强参照),可被涟漪/球轻推轻晃。
- 依赖:K6(缩放联动);与 K8 漂浮层**同框架**(可复用点精灵/粒子层)。
- 验收(⏸):睡莲叶/芦苇随水缩放、轻漂晃;边缘有"岸/边界"感、构图被框住;不抢球。

## K10 — 可见塘底层（折射透视纵深）⏸ 【开放】

- 📦 范围:水面之下加一层**静止塘底**(细石/沙纹/暗纹,renderOrder 在基调之上、水面合成之下);透过折射看到。控制台开关 + 强度参数。
- 做什么:静止塘底纹理,水面涟漪/缩放**折射这层** → **动的水面 vs 不动的塘底**产生视差 → 缩放(K6)与深度(K3)立刻"看得见"(物理上最强的纵深/缩放线索)。
- 红线:塘底是**暗纹**非亮底 → 不破"水下不压黑";浅塘不虚化;极淡 + 可调强度 + 可关 = 回纯黑塘底现状。
- 依赖:折射合成(已有);K6 缩放、K3 深度。
- 验收(⏸):透过水见极淡塘底纹;升降水位/起涟漪时塘底与水面视差明显 → 纵深强;关掉 = 纯黑塘底现状。

## K11 — 月光倒影⏸ 【开放】

- 📦 范围:水面合成加一道大柔光月光倒影(`compositeMaskFrag` 或叠加层);被高度场梯度扭动;跟 K6 缩放。控制台开关 + 强度参数。
- 做什么:一道/一团**柔亮冷白月光倒影**,被涟漪**扭动碎裂**(顺带展示波纹)、跟缩放**放缩**(参照),点题"月夜水塘"。低频柔光、强度可调,**不盖过球**。
- 依赖:高度场/合成(已有);K6 缩放。
- 验收(⏸):水面见月光倒影,涟漪扭它、水位缩放它;氛围点题;强度可调、不抢球。
